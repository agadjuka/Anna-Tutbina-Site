# Как убрать временные ограничения

### 1. Вернуть все страницы сайта:
В корне проекта удалите файл `middleware.ts` или переименуйте его (например, в `middleware.ts.off`).

### 2. Вернуть кнопки в хедере:
В файле `components/sections/header.tsx` поменяйте:
```tsx
const HIDE_NAVIGATION = true;
```
на
```tsx
const HIDE_NAVIGATION = false;
```

### 3. Доступ к сайту через админ-режим (без снятия ограничений):
Если вам нужно посмотреть сайт целиком, не снимая ограничений для всех пользователей, просто добавьте `/admin` в начало пути в адресной строке.
* Например: `site.com/admin/` (откроет главную со всеми кнопками)
* Например: `site.com/admin/tours/name` (откроет конкретный тур)

В этом режиме будут доступны все страницы и навигация в хедере, даже если включены ограничения.
